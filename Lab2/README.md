## 编译运行环境

Windows10 Ubuntu20.04

编译器版本-gcc 9.3.0

## 编译运行指南

### windows10

1. 用命令行cd到当前目录下
2. $ g++ main.cpp GameBoard.cpp
3. 将编译出a.exe文件
4. $ a.exe 或 $ a.exe -t(测试模式)

### ubuntu20.04

1. cd到当前目录下
2. $ g++ main.cpp GameBoard.cpp
3. 将编译出a.out文件
4. $ ./a.out 或 $ ./a.out -t(测试模式)
